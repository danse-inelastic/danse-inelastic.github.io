Installation

Install a DANSE INS package binary distribution:

Ubuntu 14.04

  $ # add danse.ins repository. only need to be done once for all danse.ins packages
  $ curl -s https://packagecloud.io/install/repositories/danse/ins/script.deb.sh | sudo bash
  $ # install a package. replace pkg-name with something like danse.ins.dsm or histogram
  $ sudo apt-get install pkg-name

Fedora 21

  $ # add danse.ins repository. only need to be done once for all danse.ins packages
  $ curl -s https://packagecloud.io/install/repositories/danse/ins/script.rpm.sh | sudo bash
  $ # install a package. replace pkg-name with something like danse.ins.dsm or histogram
  $ sudo yum install pkg-name

This page is for the release of utility and infrastructural software developed by the Caltech group. The release pages for the main science applications, major subpackages, and tools are below.

Flagship Applications

  • Inelastic Data Reduction
  • VNF: Virtual Neutron Facility

Major Subpackages

  • MCViNE: Monte Carlo Virtual Neutron Experiment
  • Matter: Material data structures
  • Ab initio simulation and modeling
  • Forcefield simulation and modeling
  • Parnasis: Scattering kernel generation and analysis tools
  • Scattering kernel models and BvK phonon calculations
  • Plotlib: matplotlib and mayavi wrapper for plotting scattering and dynamical functions

Tools

  • histogram
  • dsm: Data stream model
  • Inelastic data format
  • bpext: Connect boost python bindings to other bindings through PyCObject
  • numpyext: Help developers to develop bindings to numpy
  • Excitation Slicer: visualization of simulated excitation surfaces
  • Forcefield db: Various database-related classes, especially for storing forcefields
  • Translator: Translate files and dataobjects
